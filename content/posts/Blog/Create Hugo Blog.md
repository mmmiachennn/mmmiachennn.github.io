## 爲什麼要寫 blog

這是一個實現自我的小天地，就是想學厲害的同事們寫寫 blog，隨手記錄自己曾經做過的事情，希望這些事情可以通過這種方式記錄到我的腦袋中。製作 blog 的方法有很多種，Medium、Hugo、Hexo、Jekyll...等。

## 爲什麼選擇用 Hugo

當初選擇使用 Hugo 有幾個原因：

1. 同事大部分都是用這個，如果有遇到問題會比較容易找到解答（就是一個偷懶的人）。
2. 像是 Medium 這種有自己的格式的 blog，一旦遇到文章搬家是一件非常困難的事情，Hugo 是以 Markdown 的方式撰寫，Markdown 是通用格式，相較之下能稍微無痛搬家。
3. 可以找喜歡的主題或是可以自己建立主題，創造專屬風格的 blog。

## Windows 怎麼建立 Hugo？

要先安裝 Chocolately 有點像是套件管理工具，類似 python 上面常使用的 pip。
（有關 Chocolately，目前還沒有深入研究，也許之後文章更新？）
安裝 Chocolately 的方法參考此[網站](https://www.liquidweb.com/kb/how-to-install-chocolatey-on-windows/) 。

安裝 Hugo 的方法參考此篇[文章](https://blog.yowko.com/windows-10-hugo/) ，所需要的工具為 Git、golang、hugo，按照文章步驟進行就可以做出一個與文章主題一模一樣的 blog 出來。
如果像我一樣對電腦只知一的讀者們，建議你們要先瞭解 Git 之後再進行，才會知道這些指令的意義。
