Python format check (Blask,Ruff)

特性 Ruff Black
主要功能 Linter 和 Formatter（All-in-one 工具） Formatter（專注於程式碼格式化）
速度 極快（用 Rust 編寫） 相對較慢（用 Python 編寫）
配置性 較高：提供更多的配置選項，可以啟用/禁用特定的 Lint 規則。格式化風格與 Black 兼容，但在某些細節上可能有更多控制。 極低：幾乎沒有配置選項，風格規則固定。
目標 取代多個現有的 Linter 和 Formatter，提供更快速、更全面的程式碼檢查和格式化方案。 提供一個堅定的、統一的程式碼格式化風格，減少風格爭議。
語言 Rust Python
採用程度 新興但快速發展，越來越多的專案開始採用。 廣泛採用，是事實上的 Python 程式碼格式化標準之一。
功能範圍 除了格式化，還包含非常快速且全面的 Linting 功能（風格、錯誤、安全等）。 主要專注於程式碼的格式化。
