什麼是 Jupyter Lab?

Mon, Apr 12, 2021

Jupyter Notebook vs Jupyter Lab ?

我認為 Jupyter Lab 是 Jupyter Notebook 的好兄弟， 一開始我是用 Jupyter Notebook，被 Jupyter Lab 有很多附加功能吸引， 進而用 Jupyter Lab，大概除了一些小操作（可拖拉 kernel 的位置，等細到不能再細的東西）不同之外， 所有操作都大同小異，轉換零成本，只有更多優點。很推薦所有還在使用 Jupyter Notebook 的人趕快換 Jupyter Lab。

Jupyter Lab 推薦功能

主題更換

白色對常在 Coding 的人來說太刺眼了。Jupyter Lab 就有提供兩種最常見的主題顏色(黑/白)， 但是他的黑色主題配色本人實在不喜歡，所以我又搜尋了如何轉換字的顏色的方法。

  • 尋找該路徑下的 index.css 的檔案。
~/anaconda3/envs/best/share/jupyter/lab/themes/@jupyterlab/theme-dark-extension/index.css
  • 主要 code 顏色從 line 322 開始，色碼使用為 hex color。 可直接搜尋 hex color，Google 會有調色盤輸入色碼可以直接看到相對應的顏色， 或是直接點選喜歡的顏色會跳出色碼。
